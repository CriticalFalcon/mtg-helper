MTG Helper - User Guide

A shared toolkit for Magic: The Gathering players to search cards, compare decks, and improve strategy.

Helpful instructions for site visitors and contributors.

Who should use this site?

  • Casual and competitive MTG players wanting quick card data access.
  • Deck builders looking for side-by-side deck statistics.
  • Collectors tracking collection composition by format and rarity.
  • Developers or contributors exploring a live Angular sample project.

How to navigate

  1. Click Card Search to find cards by name, set, type, or price.
  2. Use Deck Compare to load two decks for matchup insights and overlap analysis.
  3. Open the Navbar to reach the home, search, and compare pages any time.
  4. Use Browser DevTools console for debug and state inspection while testing.

Contributing

  • To edit content, modify src/app/home/home.component.html.
  • Run locally with npm install and ng serve for development preview.
  • Push changes to master and run ng deploy to publish.
  • Ensure GitHub Pages is set to gh-pages branch / root folder in repository settings.

Quick troubleshooting

If old README appears online, clear browser cache and verify the pages source branch in GitHub settings, then redeploy with ng deploy.

For extension, add content in src/app/home/home.component.html and sync via standard git workflows.
